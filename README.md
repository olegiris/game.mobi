# Игра "Catch the Ball" - Олег «СТК»

Улучшенная версия игры с модульной архитектурой, визуальными эффектами и прогрессией сложности.

## Особенности

- ✅ Модульная архитектура (разделение на логические модули)
- ✅ Система частиц для визуальных эффектов
- ✅ Прогрессивное увеличение сложности
- ✅ Три уровня сложности (Легкий, Средний, Сложный)
- ✅ Сохранение рекордов в базе данных SQLite
- ✅ Улучшенный пользовательский интерфейс
- ✅ Система паузы и подтверждения выхода
- ✅ Поддержка изображений с fallback на геометрические фигуры
- ✅ Адаптивные размеры под разрешение экрана

## Установка

1. Убедитесь, что установлен Python 3.7+
2. Установите зависимости:
```bash
pip install pygame numpy
```

**Примечание**: `numpy` нужен для генерации звуковых эффектов. Если не установить, игра будет работать, но без звуков.

## Запуск

```bash
python main.py
```

## Структура проекта

```
Catch_the_Ball/
├── main.py              # Главный файл игры
```

## Управление

- **Мышь**: Движение корзины
- **Клик по кнопке паузы**: Поставить игру на паузу
- **Клик по кнопке выхода**: Выйти из игры (с подтверждением)
